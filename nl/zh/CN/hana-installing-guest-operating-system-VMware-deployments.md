# 3. 在 ESX 管理程序上安装访客操作系统（可选）
{: #install_guest_os}

如果决定使用预先安装的 ESX 管理程序，那么您负责安装 Red Hat Enterprise Linux for SAP HANA 7.4 操作系统作为基于该管理程序的访客操作系统。您需要确保部署 SAP HANA 的每个虚拟机满足 SAP HANA Tailored Data Center Integration (TDI) 关键性能指标 (KPI)。
